# 🚆 Railway Ticket System (HTML, CSS, JavaScript)

## 📌 Project Overview
This is a simple Railway Ticket Booking System web project built using HTML, CSS, and JavaScript. It allows users to add passenger details, view tickets, and cancel them using a simple interactive interface.

## ✨ Features
- 🎫 Add passenger name and destination
- 📋 View ticket list dynamically
- ❌ Cancel/Delete tickets
- 🎨 Simple and clean railway-style UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🚀 How to Run
1. Download or clone the project
2. Open the folder
3. Open `index.html` in any browser (Chrome recommended)
4. Start booking tickets

## 📂 Project Structure
- index.html (contains HTML, CSS, and JavaScript)
